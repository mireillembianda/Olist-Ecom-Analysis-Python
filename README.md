# Analyse Exploratoire de la Marketplace Olist

## Contexte
Dans ce projet, j’ai réalisé une analyse exploratoire complète d’une marketplace e-commerce brésilienne (Olist), basée sur un dataset de plus de 96 000 commandes réparties sur 7 tables.

## Objectifs
- Comprendre les performances logistiques
- Identifier les facteurs impactant la satisfaction client
- Extraire des insights business exploitables

## Outils utilisés
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Google Collab

## Analyses réalisées
- Nettoyage et fusion de datasets multi-tables
- Traitement des données temporelles (dates de commande, livraison)
- Calcul des délais de livraison et retards
- Analyse de corrélation avec la satisfaction client

## Résultats clés
- Identification d’une corrélation entre retard de livraison et mauvaise satisfaction
- Mise en évidence de tendances de croissance entre 2016 et 2018
- Identification de leviers logistiques pour réduire le churn
